
# GFX_light

Легковесная графическая библиотека для дисплеев, совместимая с Adafruit_GFX. Предоставляет базовый функционал для отрисовки текста и графических примитивов с минимальным потреблением памяти.

## Особенности

- Минимальное потребление памяти
- Встроенный шрифт 5x8 пикселей (95 символов от 32 до 126)
- Поддержка поворота дисплея (0°, 90°, 180°, 270°)
- Масштабирование текста
- Базовые графические примитивы
- Совместимость с Arduino Print (поддержка print/println)
- Оптимизирована для AVR архитектуры

## Установка

### Ручная установка
1. Скачайте архив с библиотекой
2. Распакуйте в папку `libraries` вашей Arduino IDE
3. Перезапустите Arduino IDE

## API Reference

### Основные методы

| Метод | Описание |
|-------|----------|
| `drawPixel(x, y, color)` | **Виртуальный метод, который вы должны реализовать.** Рисует один пиксель. |
| `fillRect(x, y, w, h, color)` | **Виртуальный метод.** Рисует закрашенный прямоугольник. Базовая реализация рисует через `drawPixel`, но вы можете переопределить его для аппаратного ускорения. |
| `drawFastVLine(x, y, h, color)` | Рисует вертикальную линию |
| `drawFastHLine(x, y, w, color)` | Рисует горизонтальную линию |
| `fillScreen(color)` | Заполняет весь экран цветом |
| `setRotation(r)` | Устанавливает поворот (0-3) |
| `invertDisplay(i)` | Инвертирует дисплей |

### Работа с текстом

| Метод | Описание |
|-------|----------|
| `setCursor(x, y)` | Устанавливает позицию курсора |
| `setTextSize(s)` | Устанавливает размер текста (1 = 5x8 пикселей) |
| `setTextColor(c)` | Устанавливает цвет текста |
| `setTextColor(c, bg)` | Устанавливает цвет текста и фона |
| `setTextWrap(w)` | Включает/выключает перенос текста |
| `print()` / `println()` | Вывод текста (унаследовано от Print) |
| `getCursorX()` / `getCursorY()` | Получить текущую позицию курсора |
| `width()` / `height()` | Получить размеры дисплея |
